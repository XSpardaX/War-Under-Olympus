# War Under Olympus

A Greek-mythology-themed **Vampire Survivors**-style auto-attacker.

## Project Structure

```
War Under Olympus/
│
├── Assets/                     # Unity game assets
│   ├── Art/
│   │   ├── Animations/         # Player & enemy animation controllers + clips
│   │   └── Tiles/              # Tilemap terrain assets & palette
│   ├── Audio/                  # Music & SFX
│   ├── Fonts/                  # PixelifySans + TextMesh Pro SDF
│   ├── Prefabs/
│   │   ├── Effects/            # Death particles, etc.
│   │   └── Weapons/            # Area, Directional, Spin weapon prefabs
│   ├── Scenes/                 # Game + Main Menu scenes
│   ├── Scripts/
│   │   ├── Utils/              # Damage numbers, menu manager, helpers
│   │   └── Weapons/            # Weapon base class + variants
│   ├── Settings/               # URP render pipeline settings
│   └── TextMesh Pro/           # TMP default resources
│
├── Packages/                   # Unity package manifest
├── ProjectSettings/            # Unity editor & build settings
│
├── Pygame/                     # Standalone Pygame prototype
│   ├── main.py                 # Entry point — run with: python main.py
│   ├── boons.py                # Greek-god boon system (Hades-style level-up picks)
│   ├── constants.py            # Display, tuning & colour constants
│   ├── entities.py             # Player, Enemy, Projectile, XP orb dataclasses
│   └── requirements.txt        # pip install -r requirements.txt
│
├── .gitignore
└── .gitattributes
```

## Running the Pygame Prototype

```bash
cd Pygame
pip install -r requirements.txt
python main.py
```

Controls: **WASD / Arrow keys** to move, auto-fires at nearest enemy.
Pick boons on level-up with **1 / 2 / 3**. Press **R** to restart after death.

## Opening in Unity

Open the root `War Under Olympus/` folder as a Unity project (Unity 2D URP).
The solution and `.csproj` files are auto-generated by Unity and gitignored.
